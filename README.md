### Description

This project was developed in react `v16` without the use of redux or hooks to allow anyone new to react and/or algorithms to download and modify it.

The currently implemented sorting algorithms are:
- Insertion Sort
- Bubble Sort
- Merge Sort
- Quick Sort
- Counting Sort
